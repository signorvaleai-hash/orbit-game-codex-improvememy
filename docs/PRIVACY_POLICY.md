# Privacy Policy (Template)

Last updated: <DATE>

Orbital Defense may collect limited data to provide gameplay progression and service reliability.

## Data collected

- Player profile name (user-provided)
- Gameplay stats (scores, survival time, rank, streak)
- Device identifier (local app identifier)
- Optional analytics events (if user consents)
- Optional crash diagnostics (if user consents)

## How data is used

- Save and restore player progress
- Sync leaderboard and profile
- Improve app stability and performance

## Consent controls

Users can choose:
- analytics on/off
- crash reporting on/off

These choices can be changed later in the app.

## Data sharing

Data is not sold. Third-party processors (for analytics/crash) may process data according to their policies.

## Data retention

Gameplay and leaderboard data are retained while the account/profile is active.
